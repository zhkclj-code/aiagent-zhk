## 本周学习任务

### 理论学习（每天 1-2 小时）

#### Day 1-2：Python 现代特性
- [ ] Python 3.10+ 新特性
  - `match-case` 语句（类似 Java switch）
  - 类型联合运算符 `A | B`（替代 `Union[A, B]`）
  - 更好的错误提示
- [ ] Python 异步编程
  - `asyncio` 基础
  - `async/await` 语法
  - 并发调用多个 LLM API

#### Day 3-4：LLM 基础概念
- [ ] 核心概念理解
  - Token 是什么？（类比：字符的"货币"）
  - Context Window（上下文窗口）
  - Temperature（温度参数）
  - Top P、Max Tokens 等参数
- [ ] 成本计算
  - Token 定价（GPT-3.5 vs GPT-4）
  - 如何估算 Token 数量
  - 成本优化策略

#### Day 5-7：Prompt Engineering
- [ ] 核心 Prompt 策略
  - Zero-shot Prompting
  - Few-shot Prompting
  - Chain-of-Thought (CoT)
  - 角色设定、任务拆解
- [ ] Prompt 设计技巧
  - 输出格式控制（JSON、Markdown）
  - 长文本处理（分段、摘要）
  - Prompt 注入防护
